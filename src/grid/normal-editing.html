Inline Editing is used to handle CRUD opertions. In Inline Editing mode the row itself is changed as edited row. You can start editing by double clicking a row or toolbar `Edit` button and then you can change the row values.

In this demo, inline editing is enabled by defining mode as `normal`.

Injecting Module

Grid features are segregated into individual feature-wise modules. To use editing feature, we need to inject Edit module using the Grid.Inject(Edit) method.
