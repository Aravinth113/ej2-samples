Grid supports client side exporting which allows you to export its data to Excel, Pdf and CSV formats.

In this demo, excelexport, pdfexport and csvexport items are defined in toolbar. For these toolbar items, we have defined actions in toolbarClick event to export the Grid data using the excelExport, pdfExport and csvExport methods.

Injecting Module


Grid features are segregated into individual feature-wise modules. To use exporting feature, we need to inject ExcelExport and PdfExport module using the Grid.Inject(ExcelExport, PdfExport) method.
