The NumericTextBox has options to restrict the input value between a specific range using min, max, and strictMode properties.

  • When you enable strictMode property, the value will be automatically change within a range on passing out-of-range values.
  • When you disable strictMode property, the NumericTextBox component allows out-of-range value with highlight textbox using error class to indicate the given value is wrong.
In this demo, numeric textboxes restricted between 10 to 100 through min, max properties. So you can enter the value between those ranges alone. More information on the range validation configuration can be found in this documentation section.
