TimePicker provides an option to customize the display format of the time value using format property. It accepts standard & custom date and time format strings as specified in MSDN.

Here, the time format displayed in the text box is customized with Time: custom text and 60 minutes step interval. To know more about custom time formatting, refer the Parsing and formatting section. By default, TimePicker component has formatted with `en` (English) culture.

More information about TimePicker and it's configuration can be found in this documentation section.
