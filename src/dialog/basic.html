The Dialog control used to display a message such as supplementary content like images and text, and an interactive content like form components within a web page. There are two types of dialogs:

  • Modal - The user should be work with dialog before interact with parent application .
  • Modeless - It allows to interact with parent application when the Dialog stay on the page.
In this demo, we have designed the different types of Dialog like alert, confirm and prompt.
