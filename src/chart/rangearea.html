This sample demonstrates range area series for temperature variation for a week.

In this example, you can see how to render and configure the range area type chart. You can use border, fill properties to customize the area.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.

Injecting Module

chart component features are segregated into individual feature-wise modules. To use rangearea series, we need to Injecting RangeAreaSeries module using chart.Inject(RangeAreaSeries) method.

More information on the range area series can be found in this documentation section.
