This sample visualizes the data about website visitors by using default funnel series. Datalabel shows the Information about the points.

In this example, you can see how to render funnel chart.dataLabel are used to represent individual data and its value.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.

Injecting Module

chart component features are segregated into individual feature-wise modules. To use Funnel series, we need to Injecting FunnelSeries module using chart.Inject(CandleSeries) method.
