This sample demonstrates bollinger bands indicator in the chart.

In this example, you can see how to render and configure the BollingerBands type Indicator. You can use border, fill properties to customize the area.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.

Injecting Module

Chart component features are segregated into individual feature-wise modules. To use BollingerBands Indicator, we need to Inject BollingerBands module using chart.Inject(BollingerBands) method.

More information on the BollingerIndicator can be found in this documentation section.
