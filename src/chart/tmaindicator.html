This sample demonstrates Triangle Moving Average indicator in the chart.

In this example, you can see how to render and configure the TMA Indicator. You can use border, fill properties to customize the area.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.

Injecting Module

Chart component features are segregated into individual feature-wise modules. To use TMA Indicator, we need to Inject TmaIndicator module using chart.Inject(TmaIndicator) method.

More information on the TMA Indicator can be found in this documentation section.
