This sample demonstrates pyramid chart for food comparision.

In this example, you can see how to render pyramid chart.dataLabel are used to represent individual data and its value.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.

Legends are disabled in this example, the information about it can be read using Tooltip.

Injecting Module

chart component features are segregated into individual feature-wise modules. To use pyramid series, we need to Injecting PyramidSeries module using chart.Inject(PyramidSeries) method.
