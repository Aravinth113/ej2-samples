In this example, you can see how to render and configure the range column type series. RangeColumn type charts are used for data having two values in y-axis. Those can be represented by high and low properties,which represent start and end point of columns in chart. You can use border, fill properities to customize the data appearance.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.

Injecting Module

chart component features are segregated into individual feature-wise modules. To use range column series, we need to Injecting RangeColumnSeries module using chart.Inject(RangeColumnSeries) method.

More information on the range column series can be found in this documentation section.
