Hierarchical Data

List Data

The TreeView component loads the data through the dataSource property, where the data can be either local data or remote data. In case of local data, the data structure can be hierarchical data or list data (with self-referential format i.e., mapped with id and parentID fields).

In this demo, the first TreeView was bound with hierarchical data which contains array of nested objects. And the second TreeView was bound with the list type data where the parent-child relation was referred by id and parentID mapping fields.

For more information, you can refer the Data Binding section from the documentation.
