The TreeView component allows to select multiple nodes by enabling the allowMultiSelection property.

In this demo, the TreeView is enabled with multiple selection, so you can select multiple nodes by pressing:

  • CTRL key - press the CTRL key and select desired nodes.
  • SHIFT key - select any node and by pressing shift key select another node, this selects all the nodes in-between the selected nodes.

In case of Mobile device,

  • Select a node and enable the multi selection option in the popup. On pressing the popup, you can be able to select multiple nodes.

For more information, refer to the Multi Selection section from the documentation.
